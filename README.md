## OpenDataBeta
Simple Extensible Tools for Municipal Civic Engagement, including a simple-to-deploy campaign finance visualization tool, as well as chatbot that serves as a light-touch interface for government websites.

Check out the open campaign finance data viz: https://opendatabeta.herokuapp.com/
